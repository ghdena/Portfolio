# Анализ данных для агентства недвижимости
[project](https://github.com/ghdena/Portfolio/blob/main/RE_agency/RE_agency.ipynb)

### Цель
Провести анализ объявлений о продаже жилой недвижимости в Санкт-Петербурге и Ленинградской области, с целью определения наиболее перспективных сегментов недвижимости.

### Инструменты
PostgreSQL:
- группировка данных, 
- агрегирующие функции, 
- подзапросы, 
- присоединение таблиц,
- CTE (обобщённые табличные выражения), 
- оконные функции.

### Описание проекта
В проекте определены (по времени активности объявления):
- самые привлекательные для работы сегменты недвижимости Санкт-Петербурга и городов Ленинградской области,
- сезонные тенденции, периоды с повышенной активностью продавцов и покупателей недвижимости Санкт-Петербурга и городов Ленинградской области,
- населённые пункты Ленинградской области в которых активнее всего продаётся недвижимость.
