# Парсинг вакансий и анализ рынка труда для аналитиков данных
[project](https://github.com/ghdena/Portfolio/blob/main/Parsing_hh/Parsing_hh_da.ipynb)

### Цель
Автоматизировать процесс сбора и систематизации данных с hh.ru для быстрой оценки рыночной ситуации по вакансиям аналитика данных: уровня зарплат, 
популярности форматов работы и ключевых требований работодателей.

### Инструменты
Python:
- selenium
- plotly
- pandas

### Описание проекта
В рамках проекта был создан скрипт, который имитирует действия пользователя в браузере для сбора данных с сайта hh.ru. 
Пользователь задает название города и скрипт автоматически:
1. Заполняет необходимые поля и собирает ссылки на все вакансии по специализации "аналитик данных".
2. Открывает каждую вакансию и парсит структурированную информацию.
3. Сохраняет полученные данные в Pandas DataFrame.
4. Формирует краткий отчет, позволяющий быстро оценить ситуацию на рынке труда:
     - **Количество вакансий:** общее число собранных предложений по городу.
     - **Зарплатный анализ:** типичный уровень предлагаемой заработной платы в зависимости от требуемого опыта работы.
     - **Требуемые навыки:** рейтинг самых популярных навыков.
     - **Формат работы:** диаграмма популярных форматов работы.

### Выводы и ценность
Проект представляет собой полный цикл работы с данными: от автоматизированного сбора "сырых" данных с веб-ресурса до их очистки и презентации 
в виде понятных графиков. Для анализа используется не весь массив собранных данных, скрипт служит основой для построения системы мониторинга рынка труда.

### Дальнейшее развитие проекта
В будущем функциональность скрипта может быть расширена:
1. Увеличение параметров: добавление возможности выбора не только города, но и специализации.
2. Состояние вакансий и хранение истории: реализация механизма сохранения полученных данных во внешний источник (csv/база данных) с возможностью последующего обновления. Это позволит отслеживать динамику рынка: автоматически помечать вакансии, которые ушли в архив, и добавлять новые, появившиеся с момента последнего запуска.
3. Визуализация и мониторинг: накопленные исторические данные могут стать основой для построения интерактивного дашборда (например, в DataLens или Power BI). Такой дашборд позволит в реальном времени отслеживать тренды заработных плат, изменение спроса на навыки и активность работодателей.
